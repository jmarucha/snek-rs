# snek-rs

A little side project to see what's this Rust thing is all about.

It's probably memory-safe and blazing-fast.

Terminal interface with `crossterm` so I guess it should run on anything. I tried to minimize terminal bandwidth so you can play via Dialing-up a BBN or something.
